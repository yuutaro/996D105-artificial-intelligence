## 簡単な要約

- プレースホルダー:
- この論文は Contrastive Predictive Coding を提案し、contrastive objective による表現学習を定式化している。
- InfoNCE を理解するうえで重要な基礎論文の 1 本である。
- 潜在空間で未来情報を予測しつつ、negative sample との比較で学習する枠組みを与えている。

## この発表での使い道

- プレースホルダー:
- InfoNCE がどこから来たのかを説明するときに使う。
- 一般的な自己教師あり学習の話から、現代的な contrastive learning の定式化へつなぐ役割として使う。
