## 簡単な要約

- プレースホルダー:
- この論文は、画像の欠損領域を周辺文脈から復元する inpainting を pretext task として使う、初期の自己教師あり学習 / 教師なし表現学習の代表例である。
- モデルは欠けた領域を予測することで、画像全体の意味的な構造を捉える表現を学習する。
- 現代的な contrastive learning に入る前の、pretext task 中心の時代を説明するために使いやすい。

## この発表での使い道

- プレースホルダー:
- 人手ラベルがなくても、データから補助課題を作れば学習信号を与えられる、という発想の導入に使う。
- contrastive learning に入る前の歴史的背景として使う。
